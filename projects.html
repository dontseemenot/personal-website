EMBS Research Poster Paper: End-To-End Deep Learning, Patient-Level Insomnia Classification (2021-22)
Abstract: In this paper we propose a novel deep learning based, end-to-end approach for patient-level insomnia classification with two distinct model architectures based off one-dimensional (1D) and two-dimensional (2D) versions of the computer vision Convolutional Neural Network (CNN), AlexNet. We achieved up to 94.4% and 88.9% patient-level classification accuracy via our 1D and 2D models respectively. E-Commerce Coffee Bean Shop (2021)
For my final year Computer Science project, I prototyped an e-commerce website that sold cafe related goods such as coffees, milks, meal kits, and desserts. Besides from basic functionality required for typical e-commerce websites, standout features I programmed were:

  • Cafepass - a loyalty program that offers tiers of discounts based on total amount spent
  • Recommendation System - products are suggested based on previously viewed and purchased products

I focused on building all the backend components from the ground up. My choice of technology stack was SQLite for the database, Python for the business logic, and Flask for the web framework. These were lightweight and easy to pick up technologies suitable for prototyping, while not offering too much abstraction (e.g. Django) such that I could not learn fundamentals of creating a web app. I designed the RDBMS and SQL schema to show all the relational mappings. The backend was built as a REST API, given that CRUD operations were required to be performed on the database regularly.

Tackling on this project, I was able to appreciate the logical thinking and problem solving process in delivering core functionality to the end user, hence my motivation in purusing a career in backend engineering.
